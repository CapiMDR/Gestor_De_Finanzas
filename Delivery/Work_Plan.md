## INTRODUCCIÓN

### Identificación
Este documento describe el Plan de trabajo de Software para el proyecto Gestor de Finanzas Personales. El objetivo del proyecto es construir un sistema que permita a los usuarios gestionar de forma centralizada sus ingresos, gastos, cuentas, presupuestos y recordatorios.

### Objetivo del Proyecto
Terminar el desarrollo dentro del tiempo y costo estimado, cumpliendo con los requisitos funcionales y no funcionales establecidos por el cliente y asegurando que el sistema se entregue con calidad.
